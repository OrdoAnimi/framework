# Engagement Collateral Index

**ZenCloud Global Consultants / Velocity Architecture Ecosystem**  
Classification: Internal — Portfolio Management  
Version: 1.0 — May 2026

---

## Purpose

This index tracks every engagement and planning document in the Velocity ecosystem. It defines who each document is for, when to use it, how it should be produced, and its current production status.

Use this index when preparing for a partner conversation, scoping a client engagement, or planning the next round of collateral production.

---

## Document Classification Key

| Code | Meaning |
|------|---------|
| **INT** | Internal — ZenCloud use only, not shared with partners or clients |
| **PTR** | Partner-facing — shared with consultancy partners, not end clients |
| **CLT** | Client-facing — suitable for enterprise client conversations |
| **PTR+CLT** | Suitable for both partner and client audiences |

---

## Production Format Key

| Code | Meaning |
|------|---------|
| **MD** | Source Markdown — current state |
| **PDF** | Requires export to PDF (no design work) |
| **DK** | Requires StudioSix deck production (PowerPoint or equivalent) |
| **WEB** | Suitable for website publication |
| **PDF+DK** | Requires both PDF export and deck production |
