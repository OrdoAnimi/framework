# Engagement Qualification Checklist

## Purpose

Use this checklist to decide whether a Velocity-aligned engagement should proceed, be reframed, or be declined.

## Go / No-Go Summary

| Area | Go signal | Concern | Decision |
|---|---|---|---|
| Sponsor | Named executive sponsor | No accountable owner | Go / Watch / No-go |
| Urgency | Decision or delivery issue is current | General curiosity only | Go / Watch / No-go |
| Decision problem | Clear decision-flow pain | Only asks for documents | Go / Watch / No-go |
| Stakeholder access | Access to EA, SA, PMO, business | Only one isolated contact | Go / Watch / No-go |
| Budget | Funding path exists | No budget or authority | Go / Watch / No-go |
| Data sensitivity | Data handling can be controlled | Wants confidential data in public tools | Go / Watch / No-go |
| Partner role | Partner/client ownership clear | Commercial ownership unclear | Go / Watch / No-go |
| Capacity | Fits current delivery capacity | Requires more parallel work than available | Go / Watch / No-go |

## Qualification Questions

1. What decision or delivery outcome must improve?
2. Who owns the decision?
3. Which teams are affected?
4. What artefacts exist now?
5. Which artefacts are ignored or duplicated?
6. What governance forums already exist?
7. Where is delivery blocked?
8. What data can be safely shared?
9. Is this partner-led, direct ZenCloud, or white-label?
10. What is the desired next step?

## Red Flags

- no sponsor
- no access to decision makers
- no budget path
- desire for free diagnostics
- request to use confidential data in public tools
- expectation that AI will make final decisions
- unclear partner/client ownership
- unrealistic timeline
- no acceptance criteria

## Recommended Next Step

| Qualification result | Recommended action |
|---|---|
| Strong fit | Propose FastStart or Decision Sprint. |
| Moderate fit | Run a paid discovery workshop. |
| Unclear fit | Request sponsor, scope, and data boundary clarification. |
| Poor fit | Decline or refer to better-suited provider. |
